# Sales Data Analysis Dashboard (Tableau)

## 📌 Overview
This project focuses on analyzing machine performance data using Tableau.  
The goal is to identify unhealthy machines, downtime patterns, and factory-wise performance.

---

## 🛠️ Tools Used
- Tableau Desktop

---

## 📊 Dataset
- Machine telemetry dataset loaded directly into Tableau.
- Key fields include:
  - Factory
  - Device Type
  - Unhealthy Count

---

## 🚀 Steps Followed

### 1. Data Loading
- Imported dataset directly into Tableau.
- Verified structure and fields.

### 2. Data Preparation
- Checked for missing values.
- Adjusted data types where needed.
- Prepared data for analysis inside Tableau.

### 3. Data Analysis
- Performed analysis on:
  - Factory-wise unhealthy machines
  - Device-wise downtime
- Used aggregation functions like SUM.

### 4. Dashboard Creation
- Built visualizations:
  - Bar chart → Factory vs Unhealthy
  - Bar chart → Device Type vs Downtime
- Combined into a dashboard.
- Adjusted layout for clarity.

### 5. Insights Generation
- Identified factories with higher unhealthy counts.
- Found devices contributing most to downtime.
- Compared performance across factories.

---

## 📈 Key Insights
- Certain factories show higher unhealthy machine counts.
- Specific devices contribute more to downtime.
- Helps identify operational inefficiencies.
